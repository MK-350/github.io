Project Experience

Smart Power Switch

Automotive product that limits inrush (transient) current during load startup by controlling a MOSFET switch based on real-time current feedback. Capable of driving 400A continuous DC current with short circuit protection, overcurrent protection, temperature cutoff, and battery voltage detection. Features CAN communication for data exchange.

Tech: STM32, CAN Bus, MOSFET Control, Current Sensing

Solar Trickle Charger

Low power system designed to maintain the charge of 12V lead-acid and lithium batteries using solar energy. Provides sustainable and maintenance-free solution for vehicle batteries by compensating for self-discharge and small load consumption over time.

Tech: Solar Power Management, PWM, Battery Monitoring

WiFi & GSM Cashless Device

Embedded software for cashless payment device integration with snack vending machines. Enables online payments and product selection via mobile interface. Implements MDB protocol for communication between cashless device and vending machine controller. WiFi and GSM modules provide internet connectivity to server.

Tech: WiFi, GSM, MDB Protocol, Payment Integration

MDB to RS232 Communication Board

Interface board for seamless communication between vending machine controller and credit card reader using MDB protocol. Ensures reliable data exchange and protocol conversion.

Tech: MDB Protocol, RS232, Communication Interface

Dosing Pump Controller

Software control system for chemical dosing pump valve based on input current (4-20mA). Provides output feedback for voltage and current monitoring with precise dosing control.

Tech: Analog Input Processing, 4-20mA Current Loop, PWM Control

Coffee/Tea Vending Machine

Complete embedded software solution handling all vending machine activities including touch display interface, heating control, water refill management, short circuit/overload protection, product dispensing, drink enable/disable, and PWM speed controls.

Tech: Touch Display, Temperature Control, PWM, Safety Protection

Four Card Dispenser Machine

Modified software for card dispensing functionality integrated with coin mechanism and bill acceptor. Detects credits and executes card dispensing based on user selection.

Tech: Coin/Bill Detection, Card Dispensing, User Interface

Milk Tank Cleaner & Data Logger

Software for controlling loads based on temperature monitoring and sending log data to server via 4G modem. Ensures proper temperatures and maintains comprehensive records.

Tech: Temperature Monitoring, 4G Modem, Data Logging

TFT Display Interface

TFT display integration with microcontroller featuring dynamic QR code generation and display. Provides high-quality graphical interface for various applications.

Tech: TFT Display, QR Code Generation, Graphics Processing
